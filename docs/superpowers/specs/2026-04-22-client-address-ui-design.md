# 客户地址管理 UI 页面设计

## 概述
在 `src/main/ui` 目录中实现纯 HTML + JavaScript 页面，调用后端 RESTful 接口实现客户地址的查询、新增、编辑、删除功能。

## 技术栈
- 纯 HTML + JavaScript（原生，无构建工具）
- 通过 fetch API 调用后端接口

## API 接口

### 1. 查询接口
- **URL**: `POST /client/address/query`
- **请求体**:
```json
{
  "clientNo": "客户号",
  "addressType": "地址类型（可选）",
  "pageNum": 1,
  "pageSize": 10
}
```
- **响应体**:
```json
{
  "code": "00000",
  "data": {
    "total": 100,
    "list": [
      {
        "seqNo": "序号",
        "addressType": "地址类型",
        "addressDetail": "地址详情",
        "lastChangeDate": "2024-01-01 12:00:00",
        "isMailingAddress": "Y/N",
        "isNewest": "Y/N"
      }
    ]
  }
}
```

### 2. 更新接口
- **URL**: `POST /client/address/update`
- **请求体**:
```json
{
  "clientNo": "客户号",
  "addresses": [
    {
      "seqNo": "序号（空则新增）",
      "addressType": "地址类型",
      "addressDetail": "地址详情",
      "isMailingAddress": "Y/N",
      "isNewest": "Y/N"
    }
  ]
}
```
- **响应体**:
```json
{
  "code": "00000",
  "data": {
    "clientNo": "客户号",
    "addresses": [...]
  }
}
```

## 表格列定义

| 列名 | 字段 | 说明 |
|------|------|------|
| 序号 | seqNo | 地址序号 |
| 地址类型 | addressType | 下拉选择 |
| 地址详情 | addressDetail | 文本显示 |
| 修改日期 | lastChangeDate | 格式：yyyy-MM-dd HH:mm:ss |
| 通讯地址 | isMailingAddress | Checkbox |
| 最新地址 | isNewest | Checkbox |
| 操作 | - | 编辑、删除按钮 |

## 功能流程

### 查询地址
1. 用户输入客户号
2. 点击查询按钮
3. 调用 `/client/address/query` 接口
4. 渲染地址列表

### 新增地址
1. 点击"新增地址"按钮
2. 弹出编辑弹窗（空表单）
3. 填写地址信息
4. 点击确定
5. 调用 `/client/address/update` 接口（seqNo 为空）
6. 刷新列表

### 编辑地址
1. 点击地址行的"编辑"按钮
2. 弹出编辑弹窗（填充当前数据）
3. 修改地址信息
4. 点击确定
5. 调用 `/client/address/update` 接口（seqNo 有值）
6. 刷新列表

### 删除地址
1. 点击地址行的"删除"按钮
2. 确认提示
3. 调用 `/client/address/update` 接口（del_flag='Y'）
4. 刷新列表

## 错误处理
- 接口返回错误码时，提示错误信息
- 网络错误时提示"网络异常"
- 表单校验失败时提示具体字段错误
